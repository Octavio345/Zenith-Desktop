# Zenith Desktop

Aplicacao web/PWA para gestao agricola, com diagnostico de doencas em soja por IA, clima, diario de campo, mapa de areas, estoque, atividades e perfil da fazenda.

A documentacao completa do projeto esta em:

[DOCUMENTACAO.md](./DOCUMENTACAO.md)

## Comandos principais

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Stack

- React 18
- Vite
- Firebase Authentication
- Firebase Firestore
- PWA com `vite-plugin-pwa`
- Leaflet e Leaflet Draw
- Framer Motion
- API externa de diagnostico de soja hospedada no Hugging Face
- OpenWeatherMap para clima
